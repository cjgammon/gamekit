    Class WebGPURenderer

    From-scratch WebGPU sprite renderer. Owns the device, pipeline, and buffers, and implements InstanceSink so its batcher can upload + draw.

    Browser-only (references navigator.gpu, GPU*); exported from gamekit/renderer, never the package root. Construct via create.

    Frame: beginFrame(vp) → push sprites through batcher → endFrame().
