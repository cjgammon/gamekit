    Class Rng

    Deterministic, seedable pseudo-random generator — the single source of randomness for game logic that must be reproducible and server-compatible (level/spawn generation, drops, spread). Zero-dependency and isomorphic.

    Backed by Mulberry32: a tiny, fast 32-bit-state generator with good statistical quality for games. Two Rngs with the same seed produce the same sequence on any platform, and getState/setState let you snapshot and restore the stream (e.g. to re-derive a level or sync a replay).

    For deterministic gameplay give an explicit seed; the default seed is derived from the clock, so an unseeded Rng varies run to run.
