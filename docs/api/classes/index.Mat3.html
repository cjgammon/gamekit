    Class Mat3

    Column-major 3x3 matrix for 2D transforms. Stored as a Float32Array so it can be uploaded to the GPU without copying.

    Layout (indices): [0] [3] [6] [1] [4] [7] [2] [5] [8]

    Which represents: [sxcos -sysin tx] [sxsin sycos ty] [0 0 1]
