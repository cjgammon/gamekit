    Class RenderGame

    A Game wired to the WebGPU renderer: it owns the renderer, an AssetLoader, and a RenderView, and overrides the render seam to draw the active scene each frame (interpolated by alpha).

    Browser-only. Construct with the async create (device acquisition is async); then load assets via assets, switchScene, and start.
