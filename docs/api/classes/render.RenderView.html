    StaticcullSprites

    cullSprites: boolean = true

    Skip entities whose (generously-padded) box falls outside the camera view — a large world then costs only what's on screen. Disabled automatically when the camera has no viewport (e.g. headless tests). Set false to draw all.

    StaticwarnOnZeroSize

    warnOnZeroSize: boolean = true

    Warn (once per entity) when a visible entity has zero width or height, so the classic "I added it but nothing shows" is self-explaining. Set false to silence (e.g. if you intentionally keep sized-later placeholders visible).
