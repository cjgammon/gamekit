    Class Camera

    A 2D view onto the world. Holds a center (x, y in world space), zoom, and rotation, and produces the matrices the renderer uploads each frame: viewProjection (world → clip space) for the GPU, plus worldToScreen/screenToWorld for input picking and HUD layout.

    Pure math — no DOM, no GPU types — so it stays in the isomorphic core. The server allocates one per Scene but never reads it.

    Per-frame behavior (follow, bounds clamp, shake) is advanced by update, which the owning Scene calls once per rendered frame.
